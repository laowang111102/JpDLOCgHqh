# 旅游信息管理系统 java1132

## 项目概述

旅游信息管理系统是一个基于Spring Boot和Vue的前后端分离的系统。它集成了多种技术，旨在提供旅游新闻、景区信息、美食信息、旅游线路管理等综合服务。

## 技术栈

- 前端：Vue
- 后端：Java, Spring Boot
- 数据库：MySQL
- 其他：MyBatis

## 功能模块

### 前台

- 展示旅游新闻
- 景区信息展示
- 美食信息展示
- 旅游线路展示
- 在线留言功能

### 后台管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

### 普通用户

- 旅游线路查询
- 订单信息管理

## 系统角色

- 普通用户
- 管理员
